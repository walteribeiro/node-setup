> This is a basic express project setup with typescript.

### 🏠 [Homepage](https://github.com/walteribeiro/node-setup#readme)

## Install

```sh
yarn install
```

## Build for production

```sh
yarn build
```

## Usage

### Development
```sh
yarn dev
```

### Production
```sh
yarn start
```

## Run tests

```sh
yarn test
```
